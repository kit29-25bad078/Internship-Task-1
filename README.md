# Sales Analytics Dashboard using Python

Developed a Sales Analytics Dashboard using Python, Pandas, and Matplotlib to analyze and visualize monthly sales performance across multiple products. The project processes raw sales data, calculates key business metrics, and generates insightful visual reports for decision-making.

### Key Features:

* Created and managed structured sales datasets using Pandas DataFrames.
* Calculated total monthly sales and overall revenue.
* Generated Key Performance Indicators (KPIs) including:

  * Total Revenue
  * Average Monthly Sales
  * Highest and Lowest Sales
  * Best Performing Month
* Produced statistical summaries using descriptive analytics.
* Visualized sales trends through line charts.
* Built stacked bar charts for product-wise sales comparison.
* Implemented data-driven reporting for business performance evaluation.

### Technologies Used:

* Python
* Pandas
* Matplotlib

### Outcome:

The dashboard provides a clear overview of sales performance, enabling users to identify growth trends, compare product contributions, and make informed business decisions through data visualization and analytics.
